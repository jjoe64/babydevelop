BLed - Klassenreferenz

Mit dem BLed-Widget können farbige LEDs dargestellt werden.

Sie müssen dazu die Dateien der Klasse BLed importieren (Menü: Bearbeiten -> Klasse importieren... -> [x]Grafische Klasse). Sie finden diese im Installationsverzeichnis von BabyDevelop (/opt/babyDevelop/beispiele/Qt/5_bled/bled.h).

Konstruktor

Erzeugt eine Led. 
   BLed::BLed(QWidget* parent);             // Variante 1
   BLed::BLed(Qt::QColor, QWidget* parent); // Variante 2
    Beispiel 
    ...
#include "bled.h"
...
BLed* lampe;
lampe = new BLed(this);          // Variante 1
lampe = new BLed(Qt::red, this); // Variante 2
lampe->setGeometry(10,10,30,30);
lampe->setColor("Qt::red");
lampe->show();
lampe->on();
...

Methoden von BLed

LED einschalten: 
   void BLed::on();
LED ausschalten: 
   void BLed::off();
(Ein-/Aus-)Zustand wechseln: 
   void BLed::toggle();
Farbe einstellen: 
   void BLed::setColor(const QColor &); [virtual]
    Mögliche Farben: red, green, yellow, ... und Farbcodes (siehe QColor).
Zustand ermitteln: 
   int oder bool BLed::state();
Größe einstellen: 
   void BLed::setGeometry(int x, int y, int w, int h);
    x = Abstand in Pixel von oberer, linken Ecke in x-Richtung.
    y = Abstand in Pixel von oberer, linken Ecke in y-Richtung.
    w = Breite in Pixel
    h = Höhe in Pixel; wobei gilt: h=w.
