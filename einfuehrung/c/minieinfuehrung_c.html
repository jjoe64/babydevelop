Gültigkeitsbereich von Variablen

Lokale Variablen
   - werden innerhalb von Funktionen() oder Anweisungsblöcken {} deklariert.
   - sind in anderen Funktionen bzw. außerhalb des Anweisungsblocks nicht bekannt
     bzw. werden (wenn nicht angegeben) beim Verlassen der Fumktion gelöscht.

Globale Variablen                                           Beispiel int a;
   werden außerhalb von Funktionen deklariert und haben              int main()
   "dateiweite" Gültigkeit.                                          {...

Statische Variablen
   behalten ihren Wert auch wenn ihr Gültigkeitsbereich
   verlassen wird.                                          Beispiel static int a;

Externe Variablen
   sind in anderen Dateien schon vorhanden und werden
   (wo angegeben) "programmglobal" benutzt.
   Beispiel  main.c           datei2.c
             int zahl;        extern int zahl;
             int main()       void funktion()
             {...             {...

Aus Dateien lesen

Beispiel

   FILE* dateizeiger;
   dateizeiger = fopen("datei.txt", "r");

   // Fehlerbehandlung
   if ( dateizeiger == NULL )
   {
      printf("Datei kann nicht geoeffnet werden!\n");
      getchar();
      return;
   }

   char zeile[88+1];                 // genügend Zeichen reservieren
   while(1)                          // Endlosschleife
   {
      fgets(zeile, 88, dateizeiger); // ganze Zeile auslesen
      if(feof(dateizeiger))          // Dateiende erreicht
         break;                      // Schleifenabbruch
      printf("%s", zeile);
   }

   fclose(dateizeiger);

void fgcolor(int farbe)
{
   switch (farbe)
   {
      case  1: printf("\033[30m");break; // schwarz
      case  2: printf("\033[31m");break; // rot
      case  3: printf("\033[32m");break; // grün
      case  4: printf("\033[33m");break; // gelb
      case  5: printf("\033[34m");break; // blau
      case  6: printf("\033[35m");break; // lila, magenta, violett
      case  7: printf("\033[36m");break; // tuerkies, cyan
      case  8: printf("\033[37m");break; // grau
      case  9: printf("\033[38m");break; // weiss
      case 10: printf("\033[01m");break; // hell(-farbig)
      case 11: printf("\033[02m");break; // dunkel(-grau)
      case 12: printf("\033[00m");break; // normal
      case 13: printf("\033[05m");break; // blink
      case 14: printf("\033[07m");break; // reverse
      default: printf("\nFehler: Farbcode nicht im Wertebereichs.\n"); break;
   }
}

void qcls(int tlcX, int tlcY, int brcX, int brcY)  // tlc = top-left-corner, brc = bottom-right-corner
{
   int x, y;
   for (x=tlcX; x<=brcX; x++)
   {
      for (y=tlcY; y<=brcY; y++)
      {
         pos(x,y);
         printf(" ");
      }
   }
   printf("\n");
}
